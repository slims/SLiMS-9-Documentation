protected string createTempFile(int $number_of_writes = 0, int $bytes_per_write = 0)

Creates a temporary file on the local filesystem and returns its path.

The $number_of_writes and $bytes_per_write parameters can be used to write $number_of_writes * $bytes_per_write times the character 'a' to the temporary file. All files created using this method will be deleted from the filesystem on tearDown(), i.e. after each test method was run.
