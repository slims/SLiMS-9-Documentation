static protected T slidingWindow(Engine $x, Engine $e, Engine $n, T> $class)

Sliding Window k-ary Modular Exponentiation

Based on http://www.cacr.math.uwaterloo.ca/hac/about/chap14.pdf#page=27 HAC 14.85 / http://math.libtomcrypt.com/files/tommath.pdf#page=210 MPM 7.7. In a departure from those algorithims, however, this function performs a modular reduction after every multiplication and squaring operation. As such, this function has the same preconditions that the reductions being used do.
